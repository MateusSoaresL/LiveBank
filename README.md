# 🏦 LiveBank

A simple Java-based banking console application for beginners to practice Object-Oriented Programming (OOP) and terminal interaction.

## 🚀 Features

- **Add Money:** Allows the user to deposit a specific amount into the account balance.
- **Transfer Money:** Enables the user to withdraw or transfer an amount from the current balance.
- **Exit:** Safely closes the application and terminates the process.
